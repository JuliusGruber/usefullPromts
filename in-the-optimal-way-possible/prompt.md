# In The Optimal Way Possible (ITOWP)

Sprinkle this phrase at the end of prompts you give to a coding agent:

> **"…in the optimal way possible."**

## Examples

| Before | After |
|--------|-------|
| "Refactor this function." | "Refactor this function **in the optimal way possible**." |
| "Add caching to the API layer." | "Add caching to the API layer **in the optimal way possible**." |
| "Fix the failing test." | "Fix the failing test **in the optimal way possible**." |
| "Design the schema for users and orders." | "Design the schema for users and orders **in the optimal way possible**." |

## When to Use

- On any non-trivial coding request where you want the agent to weigh trade-offs rather than ship the first solution.
- When planning architecture, refactors, or performance-sensitive code.
- Liberally — the author recommends sprinkling it across prompts, not reserving it for special cases.

## When to Skip

- One-shot mechanical edits (rename a variable, run a command) where there is no "optimal" to consider.
